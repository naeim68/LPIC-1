# ✏️ ویرایش با vi

## حالت‌ها
- **Command Mode** (پیش‌فرض)
- **Insert Mode** (با `i`)

ساخت/باز کردن فایل:
```bash
vi ./file3.txt
```

## کلیدهای مهم
| کلید | عملکرد |
|:--|:--|
| `i` | ورود به Insert |
| `Esc` | خروج از Insert |
| `:wq` | ذخیره و خروج |
| `:q` | خروج بدون تغییر |
| `:q!` | خروج اجباری |
| `o` / `Shift+o` | خط بعد/قبل + Insert |
| `yy` | کپی خط |
| `p` / `Shift+p` | Paste بعد/قبل |
| `1000p` | Paste ۱۰۰۰ بار |
| `dd` | حذف خط |
| `u` | Undo |
| `Shift+g` / `g` | آخر/ابتدای فایل |
| `:3740` | رفتن به خط ۳۷۴۰ |
| `:set nu` | شماره خطوط |

## جستجو
- `/string` به جلو، `?string` به عقب  
- `n` بعدی، `Shift+n` قبلی
